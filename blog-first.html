Tmote(TwitchTV 리모컨 사이트)

트위치 API를 이용하여 트위치 리모컨 사이트 개발(개발중)

  • Moon-Taein
  • Github-repository

● 주제

- 트위치 API를 활용하여 팔로우리스트 한눈에 보는 사이트
- 트위치 웹사이트의 불편한 부분 개선하기
 > 약한 알람 기능
 > ...

● 학습 목표

- Open API 공부하고 사용법 숙달
- Spring Boot를 사용하며 웹페이지 제작
- Bootstrap 사용하여 페이지 스타일 구성

● 개발 중점 사항

- API 정보를 request, response 데이터 형태와 타입 유의
- JSON 형태의 데이터 포맷팅

● 개발 도중 생긴 이슈

- JSONArray로 받아온 팔로우리스트 정보를 토대로 스트리머의 현재 방송 상태를 조회할때 request가 다수 발생
- request&response의 반복적인 수행으로 인해 로딩 속도가 굉장히 느림
 > 한번에 한명의 정보를 받아오기 때문에 여러번의 request가 발생
 > 파라미터에 추가로 여러명의 id를 추가해서 한번에 여러명의 정보를 받아오도록 설계
